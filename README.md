## Abra no navegador

[Petshop desenvolvido com NodeJS, TypeScript, Mustache](https://petshop-node-ts.herokuapp.com/)

### Requisitos para uso:
`npm i -g typescript nodenom ts-node`

### Rodando o projeto:
`npm run start`

### Instalação pós clone:
`npm install`
